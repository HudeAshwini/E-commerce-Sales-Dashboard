# E-commerce-Sales-Dashboard
Project Overview
This project presents an E-Commerce Sales Dashboard built in Power BI to analyze sales performance, customer behavior, and profitability. The dashboard helps businesses monitor key metrics and identify trends for better decision-making.

Key Performance Indicators (KPIs):

Total Sales Amount – Total revenue generated from orders
Total Profit – Overall profit from sales
Total Quantity Sold – Number of products sold
Average Order Value (AOV) – Average revenue per order

Dashboard Visualizations:

Sum of Amount by State – Identifies top performing states by sales
Sum of Quantity by Category – Shows which product categories sell the most
Sum of Amount by Customer Name – Highlights top customers contributing to revenue
Profit by Month – Displays monthly profit trends
Profit by Category – Compares profitability across product categories

Tools & Technologies Used:

Power BI
Data Cleaning & Data Modeling
Data Visualization

Key Insights:

Sales performance varies significantly across states.
Certain product categories contribute more to total quantity sold.
A few key customers generate a large portion of revenue.

Monthly profit trends help identify high-performing and low-performing periods.
